# VFX Gallery
![v1.0.17](https://img.shields.io/badge/version-1.0.17-brightgreen)

Десктопный инструмент для управления видео-референсами.
Просматривайте, тегируйте и организуйте видеоколлекции прямо в браузере — без облака.

## Возможности

- Галерея с автопревью и тегами
- Быстрый поиск по тегам и коллекциям
- Trim и Multicut — нарезка фрагментов
- Импорт видео по ссылке (YouTube и др.)
- Автоматические обновления

## Установка

1. Скачайте архив `VFX-Gallery-<версия>.zip` из раздела [Releases](../../releases)
2. Распакуйте в любую папку
3. Запустите `run.bat`

Python, FFmpeg и все зависимости установятся автоматически при первом запуске.

## Требования

- Windows 10 / 11
- Доступ в интернет (только для установки зависимостей и скачивания видео)
